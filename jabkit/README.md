## `JabKit`

`JabKit` offers the BibTeX and BibLaTeX handling functionality of JabRef as CLI application.

Currently, the simplest way to run `JabKit` is via JBang.
See <https://github.com/JabRef/jabref/tree/main/.jbang> for details.

It is also packed as GitHub action at <https://github.com/JabRef/jabref-action>.
