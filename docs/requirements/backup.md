# Backup
## Show library tab when reviewing backup
`req~backup.review.focus-correct-tab~1`

When the "Review Backup" dialog is shown for a library, the tab belonging to that library should be focused. After the user accepts or rejects changes and clicks OK, the correct library tab should be marked as modified — not whichever tab happens to be currently active.
