Configuration de l'aperçu des entrées

L'aperçu des entrées est créé en utilisant le même mécanisme que pour les filtres d'exportation personnalisés. En mode aperçu, une entrée est traitée en utilisant l'une des deux mises en page disponibles (vous pouvez basculer entre les deux mises en page en pressant F9) afin de produire du code HTML qui est affiché dans le panneau d'aperçu. Pour modifier l'apparence et le contenu de l'aperçu vous devez éditer les mises en page de l'aperçu en utilisant la syntaxe décrite dans le fichier d'aide portant sur les filtres d'exportation personnalisés.
